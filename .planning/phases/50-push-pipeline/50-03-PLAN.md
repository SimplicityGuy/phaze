<threat_model>
## Trust Boundaries
| Boundary | Description |
|----------|-------------|
| file-server agent → compute agent (SSH/Tailscale) | untrusted-network transport; the rsync subprocess + ssh options cross here. |
| agent → control internal API | report_pushed/report_push_mismatch (token-authed) are the only way the Postgres-free agent mutates state. |
| compute scratch dir | new on-disk state on the remote OCI box; swept at startup. |

## STRIDE Threat Register
| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-50-injection | Tampering / Elevation | rsync/ssh subprocess argv | mitigate | `create_subprocess_exec` with a fixed list argv, never a shell; remote path is `<file_id>.<ext>` (server UUID), never the untrusted filename; justified inline bandit nosec on the exec line. |
| T-50-spoof | Spoofing / Tampering | SSH host-key (TOFU/MITM) | mitigate | `-o StrictHostKeyChecking=yes -o UserKnownHostsFile=<pinned known_hosts> -o BatchMode=yes` (D-07); Tailscale authenticates the network path. |
| T-50-scratch-dos | Denial of Service | orphaned scratch from a hard-killed worker | mitigate | compute-only startup janitor full-sweeps the scratch dir; any still-needed file is re-pushed by the cron (D-14). |
| T-50-hang | Denial of Service | hung transfer pins a worker slot | mitigate | rsync `--timeout` + ssh `ConnectTimeout` + `asyncio.wait_for` outer bound; `BatchMode=yes` prevents auth-prompt hangs. |
| T-50-secret-leak | Information Disclosure | SSH key / stderr in logs | mitigate | never log the `-i` key path contents; bound stderr to a short snippet; secrets are `_FILE` SecretStr (50-01). |
| T-50-no-fallback | Tampering | missing rsync/ssh binary → silent local analysis | mitigate | FileNotFoundError → clear terminal error; NEVER fall back to local analysis (CLOUDROUTE-02 invariant). |
| T-50-SC | Tampering | npm/pip/cargo installs | accept | No new packages; rsync/openssh-client are OS binaries provisioned in Phase 51 (no language-registry surface). |
</threat_model>
