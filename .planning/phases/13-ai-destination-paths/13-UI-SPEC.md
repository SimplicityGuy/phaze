## Interaction Contracts

### Proposal Table: Destination Column

- Truncation: 40 characters with CSS `truncate` class (ellipsis). Full path on native `title` tooltip hover.
- Collision badge: rendered inline after the path text when the proposal ID is in the collision set. No click action on the badge itself.
- "No path" badge: static, no interaction.

### Collision Detection Flow

1. User navigates to proposal list page.
2. Server computes collision IDs (SQL GROUP BY) on every page load.
3. Collision IDs passed as template context to each proposal row.
4. Rows with collisions display the orange "Collision" badge in the Destination cell.
5. When user clicks "Start Execution", server re-checks collisions.
6. If collisions exist, HTMX swaps the collision block partial instead of starting execution.
7. User must resolve (reject one conflicting proposal) and retry.

### Directory Tree Preview

1. User clicks "Preview" in nav bar.
2. Server queries all approved proposals, builds tree in Python, renders page.
3. Tree nodes at depth 0-1 are open by default. Deeper nodes are collapsed.
4. "Expand All" / "Collapse All" buttons toggle all `<details>` elements via Alpine.js (`document.querySelectorAll('details').forEach(d => d.open = true/false)`).
5. No HTMX lazy loading in v1 -- full server render. Add lazy subtrees only if page load exceeds 3 seconds.

### Skip Link Update

Update skip link in `base.html` to be contextual: on `/preview/` page, skip link targets `#tree-container` instead of `#proposals-table`.

## Accessibility

| Requirement | Implementation |
|-------------|---------------|
| Collision badge semantics | `role="status"` on collision badge, `title` with full explanation |
| Tree keyboard navigation | Native `<details>/<summary>` provides Enter/Space toggle |
| Tree screen reader | Folder file counts in summary text (not hidden) |
| Skip link | Contextual: `#proposals-table` on proposals page, `#tree-container` on preview page |
| Color contrast | Orange-700 on orange-100 passes WCAG AA for text. Gray-400 on gray-100 for "No path" badge passes AA for large text (12px semibold qualifies). |
| Collision block | `role="alert"` on container for screen reader announcement |
