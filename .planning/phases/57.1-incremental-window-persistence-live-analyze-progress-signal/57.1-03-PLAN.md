<objective>
Build the single lane-agnostic HTTP surface for mid-flight progress (D-02): a new `AnalysisProgressPayload`, a new **sibling** `post_analysis_progress` handler co-located on the existing `agent_analysis.py` router, and a new `agent_client.post_analysis_progress` method. The START call carries `(analyzed=0, total=N)`; bumps carry `(analyzed=k, total=N)`. Ships **fine-only** counts (Claude's Discretion: fine-only satisfies WORK-04; coarse omitted).

Purpose: One code path, all three lanes, counter-only, no premature completion.
