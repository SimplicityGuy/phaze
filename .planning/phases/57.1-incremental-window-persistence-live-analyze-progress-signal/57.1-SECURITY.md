## Trust Boundaries

| Boundary | Description | Data Crossing |
|----------|-------------|---------------|
| Agent → control plane (new progress endpoint) | `POST /api/internal/agent/analysis/{file_id}/progress` — a NEW bearer-authenticated write surface reporting mid-flight window counts | `(fine_windows_analyzed, fine_windows_total)` ints; bearer token in header |
| analyze_file child → parent (pebble lane) | Synchronous `progress_cb` in a `run_in_process_pool` child (no event loop / no client) bridged to the parent via a `multiprocessing.Manager().Queue()` proxy | only `(int, int)` counts — no token, no client, no PII |
| analyze_file thread → loop (k8s lane) | `analyze_file` in `asyncio.to_thread`; the sync cb schedules a best-effort POST via `run_coroutine_threadsafe` | `(int, int)` counts |
| Completion write (`put_analysis`) | The authoritative completion path that stamps `analysis_completed_at` + flips `FileState.ANALYZED` | analysis aggregates + window rows |
| Proposal convergence query | `get_proposal_pending_batches` reads analysis completion state to decide eligibility | DB read; gates on `analysis_completed_at IS NOT NULL` |
