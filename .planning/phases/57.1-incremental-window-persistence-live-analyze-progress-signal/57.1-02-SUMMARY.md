# Phase 57.1 Plan 02: Completion Discriminator & Convergence-Gate KEY-RISK Fix Summary

**Added `analysis_completed_at` (migration 028), stamped only in the existing `put_analysis` completion branch, and tightened the proposal convergence gate to require it — so a partial in-flight analysis row (written at START by D-03) can never leak into `generate_proposals` with NULL aggregates.**

## Accomplishments

- **KEY RISK closed (T-57.1-10):** `get_proposal_pending_batches` no longer treats bare `exists(AnalysisResult)` as "analysis complete". It now requires `analysis_completed_at IS NOT NULL`, so a `METADATA_EXTRACTED` file carrying a partial START row (NULL bpm, `fine_windows_analyzed < total`, `completed_at` NULL) returns in ZERO batches — proven by regression, with a positive control showing the same file appears once completion is stamped.
- **Completion discriminator landed FIRST (wave 2):** migration 028 + `AnalysisResult.analysis_completed_at` + the `func.now()` stamp ship before any partial row is ever written (waves 3+), so the safety net precedes the regression surface.
- **Spoofing mitigation locked (T-57.1-12):** `analysis_completed_at` is server-set only — absent from `AnalysisWritePayload` and `_ANALYSIS_COLUMN_FIELDS`; a body carrying it is a 422. A client cannot forge completion.
- **Search invariant locked (T-57.1-11):** a bpm-filtered search already excludes a NULL-bpm partial row (NULL comparison → unknown); added an assertion to lock it with no code change to `search_queries.py`.

## Decisions Made

- **Column over zero-migration tighten** (RESEARCH Q4): the explicit `analysis_completed_at` discriminator is intention-revealing and gives Phase 58 a crisp `row exists + completed_at IS NULL = in-flight` flag, with no risk of dropping a legitimate `METADATA_EXTRACTED`-with-complete-analysis file.
- **Server-set-only stamp**: a sibling `update(AnalysisResult)` in the existing completion branch keeps the empty-body PUT path (`dumped` falsy) from stamping anything, exactly mirroring the ANALYZED-flip guard; excluding it from the wire payload blocks forgery.
