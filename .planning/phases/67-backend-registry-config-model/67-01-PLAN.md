## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| operator config file → config layer | Operator-authored `backends.toml` values (endpoint_url, `*_file` paths) parsed into typed models |
| secret mount file → config layer | Inline `*_file` paths point at operator-mounted secret files read at construction |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-67-01-01 | Tampering | BucketConfig.endpoint_url (SSRF surface) | mitigate | Lift `_validate_s3_endpoint_url` (config.py:597-613) as a per-bucket field_validator: http(s) scheme + non-empty netloc required at construction |
| T-67-01-02 | Information disclosure | resolved kube/S3/agent secrets | mitigate | Resolved secret fields typed `SecretStr` so accidental interpolation prints `**********` |
| T-67-01-03 | Tampering / Info disclosure | inline `*_file` path read | mitigate | Fail-fast ValueError on unreadable path; never echo file contents in the error (operator-controlled single-user tool — not a privilege boundary, still fail-closed) |
| T-67-01-04 | Denial of service | out-of-range rank/cap | mitigate | Bounded integer fields reject out-of-range values at construction (fail-fast posture) |
| T-67-01-SC | Tampering | package installs | accept | Zero new dependencies this phase (RESEARCH: stdlib tomllib + pinned pydantic only); no install task exists |
</threat_model>
