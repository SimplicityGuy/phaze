       **async def search_tracklist(ctx: dict[str, Any], file_id: str) -> dict[str, Any]:**

       - Load Tracklist by id

       - Close scraper
       - Return `{"tracklist_id": tracklist_id, "tracks_found": len(tracks), "version": version_number}`
