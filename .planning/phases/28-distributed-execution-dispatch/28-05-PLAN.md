- `Job` has `meta: dict[Any, Any]` attribute (default `{}`).

That works because if a path-resolve fails the current_step is "copy" (matches operator intuition — "the copy didn't happen"), and if sha256 fails current_step is "verify" (matches CONTEXT D-07 example).
