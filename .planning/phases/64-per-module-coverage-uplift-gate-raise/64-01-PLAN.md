<objective>
Build the per-module coverage floor enforcement machinery: a stdlib-only Python script
`scripts/coverage_floor.py` that reads `coverage json` output and exits non-zero if any
tracked `phaze/**` module falls below the uniform 85% floor, plus its own unit test.

This is the net-new engineering of the phase (coverage.py has NO native per-file
`fail_under` — D-02). The script is CI-load-bearing, so it gets its own tests (Phase 63
precedent: `scripts/classify-changed-files.sh` is tested by `tests/shared/test_change_gate.py`).

Purpose: Enforce COV-01's per-module floor so one module cannot silently rot while the
aggregate stays high (D-01 rationale). Zero new dependencies (D-02 discretion, RESEARCH
Package Legitimacy Audit — stdlib `json`/`sys`/`pathlib` only).
Output: `scripts/coverage_floor.py` + `tests/shared/test_coverage_floor.py`.
</objective>
