  <action>
    **Create `src/phaze/services/discogs_matcher.py`:**

    1. `DiscogsographyClient` class following AudfprintAdapter pattern:
       - `__init__(self, base_url: str, timeout: float = 30.0)` -- creates `httpx.AsyncClient(base_url=base_url, timeout=timeout)`
       - `async def search_releases(self, query: str, limit: int = 10) -> list[dict[str, Any]]` -- GET `/api/search` with params `{"q": query, "types": "release", "limit": limit}`. Catch `httpx.ConnectError` and `httpx.TimeoutException`, log warning, return empty list. Otherwise `raise_for_status()` and return `data.get("results", [])`.
       - `async def close(self) -> None` -- calls `self._client.aclose()`

    2. `compute_discogs_confidence(track_artist: str, track_title: str, discogs_name: str, discogs_relevance: float) -> float`:
       - Combine `f"{track_artist} {track_title}".lower().strip()` as track_query
       - Use `rapidfuzz.fuzz.token_set_ratio(track_query, discogs_name.lower().strip()) / 100.0` for string_sim
       - Blend: `confidence = (string_sim * 0.6 + discogs_relevance * 0.4) * 100.0`
       - Return `round(confidence, 1)`, clamped to 0-100 range

    3. `async def match_track_to_discogs(client: DiscogsographyClient, track: TracklistTrack) -> list[dict[str, Any]]`:
       - If track.artist is None/empty or track.title is None/empty, return empty list (D-02)
       - Build query: `f"{track.artist} {track.title}"`
       - Call `client.search_releases(query, limit=10)`
       - Score each result with `compute_discogs_confidence`
       - Sort by confidence DESC, return top 3 as dicts with keys: discogs_release_id, discogs_artist (from result "name" field -- parse artist from "Artist - Title" format if possible), discogs_title, discogs_label (from metadata if available), discogs_year (from metadata.year), confidence
       - Note: discogsography results have `name` field which is the full release name. Parse it best-effort but store the full `name` as discogs_title if parsing fails.

    **Create `src/phaze/tasks/discogs.py`:**

    `async def match_tracklist_to_discogs(ctx: dict[str, Any], *, tracklist_id: str) -> dict[str, Any]`:
    - Get async_session from ctx
    - Load tracklist by ID, get latest_version_id, load tracks for that version
    - Filter eligible tracks (both artist AND title non-null and non-empty per D-02)
    - For each eligible track: delete existing DiscogsLink rows where track_id matches AND status="candidate" (preserve "accepted" links per pitfall 3)
    - Create DiscogsographyClient using `settings.discogsography_url` from config
    - Use `asyncio.Semaphore(settings.discogs_match_concurrency)` to limit concurrent requests
    - Use `asyncio.gather` to match all eligible tracks concurrently (bounded by semaphore)
    - For each match result, create DiscogsLink instances and add to session
    - Commit, close client
    - Return `{"tracklist_id": tracklist_id, "tracks_matched": N, "tracks_skipped": M, "candidates_created": C}`

    - In `shutdown`, close it: call `await ctx["discogs_client"].close()` if present
