Create `src/phaze/services/ingestion.py` with these functions:

1. `normalize_path(path: str) -> str`:
   - `return unicodedata.normalize("NFC", path)`

2. `compute_sha256(file_path: Path) -> str`:
   - Use `hashlib.sha256()` with `HASH_CHUNK_SIZE` chunked reads per D-07
   - Synchronous function -- caller wraps in `asyncio.to_thread()` if needed
   - Return hex digest string

3. `classify_file(filename: str) -> FileCategory`:
   - Get suffix via `Path(filename).suffix.lower()`
   - Look up in `EXTENSION_MAP`, default to `FileCategory.UNKNOWN`

4. `discover_and_hash_files(scan_path: str, batch_id: uuid.UUID) -> list[dict]`:
   - Walk with `os.walk(scan_root, followlinks=False)` per research anti-pattern advice
   - For each file: classify, skip UNKNOWN, NFC-normalize path and filename, compute sha256, get file_size
   - Build dict with keys: `id` (uuid4), `sha256_hash`, `original_path` (NFC normalized), `original_filename` (NFC normalized), `current_path` (same as original_path), `file_type` (extension without dot, lowercase), `file_size`, `state` (FileState.DISCOVERED), `batch_id`
   - Wrap unreadable files in try/except OSError and skip them (log warning)
   - Return list of dicts

5. `async def bulk_upsert_files(session: AsyncSession, records: list[dict], batch_size: int = BULK_INSERT_BATCH_SIZE) -> int`:
   - Use `itertools.batched(records, batch_size)` (Python 3.12+)
   - For each batch: `await session.execute(insert(FileRecord).on_conflict_do_update(index_elements=["original_path"], set_={...}), list(batch))` -- on conflict update sha256_hash, file_size, state, batch_id, file_type (per D-08 resumability)
   - NOTE: PostgreSQL ON CONFLICT with bulk executemany requires using `insert().on_conflict_do_update()` with `index_elements=["original_path"]`. If the executemany pattern does not support on_conflict, fall back to: query existing paths as a set, filter new records, bulk insert only truly new ones, then bulk update existing ones separately.
   - `await session.commit()` after each batch
   - Return total count

6. `async def run_scan(scan_path: str, batch_id: uuid.UUID, session_factory: async_sessionmaker) -> None`:
   - Create ScanBatch record with status RUNNING
   - Run `discover_and_hash_files` via `asyncio.to_thread()` (per D-06, scanning is sync)
   - Update ScanBatch.total_files
   - Call `bulk_upsert_files` to persist
   - Update ScanBatch status to COMPLETED (or FAILED on exception with error_message)
   - This is the top-level orchestrator called from the router

All functions must have type hints. Use `import logging` and `logger = logging.getLogger(__name__)` for logging.

Commit after implementing the service.
