## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| operator HTTP POST -> deepen endpoint | path-param file_id (uuid) crosses into a queue-producing action |
| deepen endpoint -> SAQ broker | re-enqueues a process_file job that an agent worker will execute |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-44-07 | Spoofing / Misrouting | re-enqueue hitting the consumer-less default queue (Phase-30 incident) | mitigate | Resolve via enqueue_router.resolve_queue_for_task("process_file", ...); NoActiveAgentError returns a fragment without enqueuing — no fall-through to default queue; test 4 enforces |
| T-44-08 | Tampering | file_id-only payload dead-lettering under extra="forbid" (v4.0.8 incident) | mitigate | Funnel through enqueue_process_file which builds the COMPLETE ProcessFilePayload; test 2 asserts the full payload |
| T-44-09 | Denial of Service | repeated deepen clicks flooding the queue | mitigate | Deterministic process_file:<file_id> key dedups an in-flight repeat to a no-op (D-05); test 3 asserts the key |
| T-44-10 | Tampering / Injection | path-param file_id | mitigate | Typed uuid.UUID path param (FastAPI 422 on malformed); scalar_one_or_none -> not-found fragment, never raw SQL or 500 |
| T-44-SC | Tampering | npm/pip/cargo installs | mitigate | No new packages installed; nothing to audit. |
