2. **`proposed_path` unused in execution** — Files always rename in-place (`source.parent / proposed_filename`). `settings.output_path` and `RenameProposal.proposed_path` are defined but never consumed. (EXE-01)
3. **`FileMetadata` model is dead code** — Table exists in migration 001, model exported, but nothing writes to it (mutagen not integrated). LLM context is filename-only. (AIP-01, v2 META-01)
4. **`.opus` extension gap** — `.opus` in analysis `_MUSIC_FILE_TYPES` but not in ingestion `EXTENSION_MAP`. `.opus` files skipped at scan, never reach analysis. (ING-05)
