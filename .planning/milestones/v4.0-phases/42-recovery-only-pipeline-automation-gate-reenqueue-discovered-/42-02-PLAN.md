| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-42-07 | Tampering | force=True manual recovery while jobs are live | accept | force bypasses only the no-op short-circuit, not the per-item dedup; any live item collapses to a skipped no-op — forcing is safe by construction (backstop, ref T-42-05). |
| T-42-04 | Denial of service | startup recovery aborting boot | mitigate | The boot-time recover call stays inside the broad try/except + logger.exception; a recovery failure is logged and boot continues. |
