- `from saq.web.starlette import saq_web`

  sets module-global `ROOT_PATH = root_path` (`saq/web/starlette.py:135-138`). It NEVER constructs a

  `/`, `/queues/{queue}`, `/queues/{queue}/jobs/{job}`, `/api/queues`, `/api/queues/{queue}`,

  `Mount("/static", ...)`, `/health`.
- `/api/queues` => `{"queues": [await q.info() for q in QUEUES.values()]}`; the rendered `/` HTML
  embeds asset links as `{root_path}/static/...` i.e. `/saq/static/...`.
