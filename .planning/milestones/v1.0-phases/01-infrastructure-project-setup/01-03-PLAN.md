**.github/workflows/ci.yml** -- Main entrypoint:
- Triggers: `push` (all branches), `pull_request` (all branches)
- Concurrency group: `ci-${{ github.ref }}` with `cancel-in-progress: true` (only on PRs, not pushes to main)
- Jobs that call reusable workflows:
  - `quality`: `uses: ./.github/workflows/code-quality.yml`
  - `test`: `uses: ./.github/workflows/tests.yml` (needs: quality)
  - `security`: `uses: ./.github/workflows/security.yml`

**.github/workflows/code-quality.yml** -- Reusable workflow (`on: workflow_call`):
- Runs on `ubuntu-latest`
- Steps:
  1. Checkout code
  2. Install uv (use `astral-sh/setup-uv@v4` or latest)
  3. Set up Python 3.13 (use `actions/setup-python@v5`)
  4. Install dependencies: `uv sync`
  5. Run pre-commit: `uv run pre-commit run --all-files`
- Cache pre-commit environments for speed

**.github/workflows/tests.yml** -- Reusable workflow (`on: workflow_call`):
- Runs on `ubuntu-latest`
- Services: PostgreSQL 16 container for integration tests
  ```yaml
  services:
    postgres:
      image: postgres:16-alpine
      env:
        POSTGRES_USER: phaze
        POSTGRES_PASSWORD: phaze
        POSTGRES_DB: phaze_test
      ports:
        - 5432:5432
      options: >-
        --health-cmd "pg_isready -U phaze"
        --health-interval 5s
        --health-timeout 5s
        --health-retries 5
  ```
- Steps:
  1. Checkout code
  2. Install uv
  3. Set up Python 3.13
  4. Install dependencies: `uv sync`
  5. Run pytest with coverage: `uv run pytest --cov=phaze --cov-report=xml --cov-report=term-missing`
  6. Upload coverage to Codecov using `codecov/codecov-action@v4` with:
     - `flags: unittests`
     - `disable_search: true`
     - `file: ./coverage.xml`
- Environment variable: `DATABASE_URL=postgresql+asyncpg://phaze:phaze@localhost:5432/phaze_test`

**.github/workflows/security.yml** -- Reusable workflow (`on: workflow_call`):
- Runs on `ubuntu-latest`
- Steps:
  1. Checkout code
  2. Install uv
  3. Set up Python 3.13
  4. Install dependencies: `uv sync`
  5. Run pip-audit: `uv run pip-audit` (add `pip-audit` to dev dependencies if not present, or use `uvx pip-audit`)
  6. Run bandit: `uv run bandit -r src/ -x tests -s B608`
  7. Run Semgrep: use `semgrep/semgrep-action@v1` or `uvx semgrep scan --config auto src/`
  8. Run TruffleHog: use `trufflesecurity/trufflehog@main` action with `--only-verified` flag

**.codecov.yml** -- Root-level Codecov configuration per CLAUDE.md:

```yaml
        target: 80%
        threshold: 5%
```
