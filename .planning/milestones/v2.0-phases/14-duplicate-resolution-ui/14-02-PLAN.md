    **Router: `src/phaze/routers/duplicates.py`**

    Create router following proposals.py pattern exactly. Register `filesizeformat` Jinja2 filter (value in bytes -> "X.X MB" format). Endpoints:

    1. `GET /duplicates/` -- list_duplicates(request, page=Query(1,ge=1), page_size=Query(20,ge=10,le=100), session=Depends(get_session))
       - Call find_duplicate_groups_with_metadata(session, limit=page_size, offset=(page-1)*page_size)
       - Call get_duplicate_stats(session), count_duplicate_groups(session)
       - Call score_group(group) for each group
       - Build Pagination(page=page, page_size=page_size, total=total)
       - context: request, groups, stats, pagination, current_page="duplicates"
       - HTMX: return "duplicates/partials/group_list.html" partial
       - Full: return "duplicates/list.html"

    2. `GET /duplicates/{group_hash}/compare` -- compare_group(request, group_hash, session)
       - Fetch single group by hash using find_duplicate_groups_with_metadata with appropriate filter
       - Call score_group(group)
       - Return "duplicates/partials/comparison_table.html" with group context
       - Also compute best-per-column: for each numeric column (file_size is SMALLEST = best, bitrate is HIGHEST = best, duration is LONGEST = best), track the best file id. Pass as best_values dict to template.

    3. `POST /duplicates/{group_hash}/resolve` -- resolve_group_endpoint(request, group_hash, canonical_id=Form(...), session)
       - Call resolve_group(session, group_hash, canonical_id)
       - Call get_duplicate_stats(session)
       - Return "duplicates/partials/resolve_response.html" with stats, group_hash, resolved_count, resolved_file_states (list of {id, previous_state} for undo), canonical_id

    4. `POST /duplicates/{group_hash}/undo` -- undo_resolve_endpoint(request, group_hash, file_states=Form(...), session)
       - Parse file_states JSON from form data
       - Call undo_resolve(session, file_states)
       - Rebuild group data (re-fetch and score)
       - Call get_duplicate_stats(session)
       - Return the group card HTML + OOB stats update

    5. `POST /duplicates/resolve-all` -- bulk_resolve(request, page=Form(1), page_size=Form(20), session)
       - Fetch all groups on current page, score each, resolve each with auto-selected canonical
       - Collect all resolved_file_states for bulk undo
       - Return "duplicates/partials/bulk_resolve_response.html" with stats + bulk toast

    6. `POST /duplicates/undo-all` -- bulk_undo(request, file_states=Form(...), session)
       - Parse bulk file_states JSON, call undo_resolve
       - Re-fetch groups and return full group list partial + stats OOB
