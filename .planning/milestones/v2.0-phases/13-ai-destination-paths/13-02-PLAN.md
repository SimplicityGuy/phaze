    Create `src/phaze/services/collision.py` with:

    1. **`TreeNode` dataclass:**
    ```python
    from dataclasses import dataclass, field

    @dataclass
    class TreeNode:
        name: str
        children: dict[str, "TreeNode"] = field(default_factory=dict)
        files: list[str] = field(default_factory=list)
        file_count: int = 0
    ```

    2. **`detect_collisions(session: AsyncSession) -> list[tuple[str, int]]`** per D-07:
    - SQL query: `SELECT concat(proposed_path, '/', proposed_filename) AS dest, count(*) AS cnt FROM proposals WHERE status = 'approved' AND proposed_path IS NOT NULL GROUP BY dest HAVING count(*) > 1`
    - Use `sqlalchemy.func.concat` and `func.count`
    - Import `RenameProposal`, `ProposalStatus` from `phaze.models.proposal`
    - Return list of `(full_path, count)` tuples

    3. **`get_collision_ids(session: AsyncSession) -> set[str]`**:
    - Call `detect_collisions` to get collision paths
    - For each collision path, query proposal IDs that match that full destination
    - Return set of string UUIDs for template rendering (used by proposal row to show collision badge)
    - Query: `SELECT id FROM proposals WHERE status = 'approved' AND proposed_path IS NOT NULL AND concat(proposed_path, '/', proposed_filename) IN (:paths)`

    4. **`build_tree(proposals: list[RenameProposal]) -> TreeNode`**:
    - Create root `TreeNode(name="output")`
    - For each proposal: if `proposed_path` is None, append `proposed_filename` to `root.files`; otherwise split path by `/`, traverse/create child nodes, append filename to leaf node's files
    - Call `_count_files(root)` to compute recursive counts
    - Return root

    5. **`_count_files(node: TreeNode) -> int`** (private helper):
    - Recursively count `len(node.files)` plus children's counts
    - Set `node.file_count` at each level
    - Return total count

    All functions must have full type annotations (mypy strict). Use `from __future__ import annotations` at top.

For `detect_collisions` and `get_collision_ids`, use an in-memory async session or mock the session appropriately following existing test patterns.
