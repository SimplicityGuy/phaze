    **Write maps** (reverse of read maps in metadata.py):

    _WRITE_ID3_MAP = {"artist": TPE1, "title": TIT2, "album": TALB, "year": TDRC, "genre": TCON, "track_number": TRCK}

    **`write_tags(file_path: str, tags: dict[str, str | int | None]) -> None`** (per D-10, synchronous):
    - Open with `mutagen.File(file_path)`, raise ValueError if None
    - If `audio.tags is None`, call `audio.add_tags()` (Pitfall 4)
    - Detect format: `isinstance(audio.tags, ID3)` for MP3, `isinstance(audio, MP4)` for M4A, else Vorbis
    - ID3: Use frame class constructors `frame_cls(encoding=3, text=[str(value)])` (Pitfall 1)
    - MP4: Use atom keys, track_number as `[(int(value), 0)]` tuple (Pitfall 2)
    - Vorbis: Direct key assignment `audio[key] = [str(value)]`
    - Skip None values. Call `audio.save()`

    **`verify_write(file_path: str, expected: dict[str, str | int | None]) -> dict[str, dict[str, str | None]]`** (per D-08):
    - Re-read with `extract_tags(file_path)` from existing metadata service
    - For each field in expected (skip None): normalize both to NFC with `unicodedata.normalize("NFC", str(val))` (Pitfall 3)
    - Return dict of discrepancies: `{field: {"expected": exp, "actual": act}}`
    - Empty dict means perfect write

    **`async def execute_tag_write(session: AsyncSession, file_record: FileRecord, proposed_tags: dict[str, str | int | None], source: str) -> TagWriteLog`**:
    - Gate: if `file_record.state != FileState.EXECUTED`, raise ValueError per D-03 — "Only executed files can have tags written"
    - Use `file_record.current_path` (NEVER original_path — Pitfall 5)
    - Read before_tags: `extract_tags(file_record.current_path)` -> serialize to dict (artist, title, album, year, genre, track_number)
    - Try `write_tags(file_record.current_path, proposed_tags)`
    - On success: `discrepancies = verify_write(file_record.current_path, proposed_tags)`
    - Status: `TagWriteStatus.DISCREPANCY` if discrepancies, else `TagWriteStatus.COMPLETED` (per D-09)
    - On exception: status = `TagWriteStatus.FAILED`, error_message = str(e)
    - Create TagWriteLog entry (per D-11): before_tags, after_tags=proposed_tags, source, status, discrepancies, error_message
    - Add to session, flush, return the log entry
