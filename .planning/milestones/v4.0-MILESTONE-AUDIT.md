## End-to-End Flow Traces

**✓ Flow 1 — File dropped on file-server → admin dashboard**
Watcher → debouncer settle → poster → `POST /api/internal/agent/files` (LIVE sentinel) → composite-UQ upsert → `extract_file_metadata` on `phaze-agent-<id>` → metadata + fingerprint POSTed back. All 9 hops wired. Live UAT passed (Phase 27-HUMAN-UAT.md).

**✓ Flow 2 — Admin-triggered scan → progress card**
`POST /pipeline/scans` → scan_root membership check → `scan_directory` enqueue → 500-record-chunk upserts → `PATCH /api/internal/agent/scan-batches/{id}`. 5 hops wired.

**⚠ Flow 3 — Approve batch → per-agent execution → SSE progress**
Approval → `get_approved_proposals_grouped_by_agent` → `exec:{batch_id}` Redis hash seeded → `execute_approved_batch` per (agent, chunk) → per-proposal write-ahead `ExecutionLog` → terminal `post_exec_batch_progress` HINCRBY → unified SSE. **Degraded edge:** the per-agent revoked-agents breakdown is silently suppressed on first render (WR-03 — see flows gap above). Aggregate count surfaces correctly.

**✓ Flow 4 — Agent heartbeat → /admin/agents liveness**
SAQ CronJob `heartbeat_tick` every 30s → `POST /api/internal/agent/heartbeat` → `Agent.last_seen_at` updated → `/admin/agents` reads + classifies + HTMX 5s poll. 5 hops wired.

**✓ Flow 5 — TLS + auth bootstrap (agent startup)**
`entrypoint.py` → `ensure_certs_present` → operator copies CA → `construct_agent_client` CA-file fail-fast → `PhazeAgentClient(verify=...)` → `httpx.AsyncClient(verify=...)`. 5 hops wired. Real two-host hardware smoke deferred by operator decision.
