direct `"FileRecord"`-textual RHS. Two real evasions are reproduced:
(a) `proposal.file.state = "approved"` — the EXACT chained-attribute shape Plan 01 deleted from
`update_proposal_status` — has an `ast.Attribute` base, so no scanner fires;

permanently-encoded RED mutation cases for the missed shapes, each mutation-verified per this project's
`feedback_mutation_test_guard_tests` lesson (a GREEN guard proves nothing — break the scanner, watch
the new case go RED, restore, watch it go GREEN).

Purpose: give the anti-drift guard real teeth against the actual deleted-cascade shapes.
Output: broadened scanners + two encoded RED cases; all real-source guards and existing GREEN
false-positive cases still pass.

TESTS ONLY — no production source changes. The three scanned source files are read for reference,
never edited.
